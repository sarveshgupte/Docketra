## 2024-05-04 - [Visual Feedback for System Clipboard API]
**Learning:** Adding immediate, visual feedback alongside standard browser `navigator.clipboard.writeText()` calls improves the overall interactive experience and helps users quickly realize an action succeeded. Providing `aria-live="polite"` to the element announcing the feedback makes the experience accessible.
**Action:** When adding clipboard interactions, include a temporary "Copied!" state and notify screen readers using ARIA live regions. Ensure the temporary state clears (e.g. after 2 seconds) so the user can easily re-copy if needed.
