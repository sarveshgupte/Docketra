## 2024-05-14 - Accessible Generic Action Buttons
**Learning:** For generic action buttons like "Mark all read" that rely heavily on their visual context for meaning, simply using an \`aria-label\` might not always be preferred if you want screen readers to have distinct announcement behavior while keeping text hidden visually without relying on untrusted custom CSS. Using standard \`sr-only\` classes defined natively within Tailwind is a clean and bulletproof way to inject contextual phrasing for screen readers (e.g. "Mark all notifications as read") alongside the `aria-hidden="true"` visible text.
**Action:** When creating text buttons for generic actions within lists or localized contexts, prefer separating the visible text inside an \`aria-hidden="true"\` element and providing specific context with a visually hidden \`<span className="sr-only">\` child. Ensure that \`sr-only\` is part of the project's styling system (it is a standard Tailwind utility).
