💡 What: Added `aria-hidden="true"` to literal icon characters (like `✓` and `✕` / `×`) inside buttons.
🎯 Why: Prevents screen readers from confusingly reading literal character names (like "multiplication X") aloud when an `aria-label` already exists.
📸 Before/After: Visuals remain unchanged; auditory experience is streamlined.
♿ Accessibility: Improves screen reader clarity and removes redundant text readouts.
